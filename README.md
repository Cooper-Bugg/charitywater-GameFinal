# Water Can Catcher 

**[Play the Game Here](https://cooper-bugg.github.io/charitywater-GameFinal/)** 

A fun browser-based game created to raise awareness for charity: water's mission to bring clean water to people in need.

## About

Water Can Catcher is an interactive game where players catch falling water cans while avoiding obstacles. The game features multiple difficulty levels and aims to engage users with charity: water's cause in an entertaining way.

## How to Play

1. Open `index.html` in your web browser
2. Select your difficulty level (Easy, Normal, or Hard)
3. Click "Start Game" to begin
4. Catch the good water cans to increase your score
5. Avoid the bad cans
6. Reach the goal before time runs out!

## Features

- Three difficulty levels
- Score tracking
- Time-based gameplay
- Responsive design
- charity: water branding and links

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
